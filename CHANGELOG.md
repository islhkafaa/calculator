# Changelog

All notable changes to this project will be documented in this file.

## [0.5.0] - 2026-02-24

### Added
- **Core Operations**: Implemented Add, Subtract, Multiply, Divide, and Percent.
- **Advanced Operations**: Added Power (`^`) and Square Root (`√`).
- **History System**: Built a state-persistent history panel with Swipe-to-Delete functionality.
- **Expressive UI**:
    - Animated shape-morphing buttons (Circle to RoundedRect).
    - Dynamic gradient backgrounds that respond to input operations.
    - Glassmorphic borders for buttons in Dark Mode.
- **UX Refinements**:
    - Contextual haptics (different feedback for numbers vs. operators).
    - Smooth layout transitions for screen rotation.
    - High-precision math engine using `BigDecimal`.

### Improved
- Button grid layout for better spacing and breathability.
- Animated display transitions for primary and secondary text.
